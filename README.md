# GlobalMart E-commerce Platform

A comprehensive e-commerce solution with dual-language support (English/Malay), ToyibPay integration, and Google Sheets backend.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Dual Language Support**: English and Bahasa Malaysia
- **Complete User System**: Registration, login, and member management
- **Admin Dashboard**: Full CRUD operations for users, products, and orders
- **ToyibPay Integration**: Secure payment processing
- **Google Sheets Backend**: Data storage without complex database setup
- **Activity Logging**: Comprehensive logging system
- **SEO Optimized**: SEO-friendly copywriting and structure
